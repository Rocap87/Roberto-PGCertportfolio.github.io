Class diagram

In unit 3 and 5 I had to create an object model based on which I developed a class diagram.

Object design / class diagram

The pratical activity in unit 3 was to create an object model representing a supermarket by considering some or all of the following classes: Staff, Products, Customers, Online orders, Loyalty schemes. I decided to use all classes and develope them in an UML class diagram. In my class diagram I applied the concepts of aggregation and composition to show the relationship between the classes.

Python phonebook

This is a Python programming challenge to assess the ability to design an algorithm to solve a problem. The first part of the assignment consisted in choosing the application to develop during the assignment and define the objective and purpose of the selected application, outline the instructions of how the application is to be executed, define and show the data structures used in the application, describe the algorithms using pseudocode, identify the checkpoints to test and outline a test plan indicating the range of data that will be used and outline the expected test results.

The second part of the assignment focuses on developing a Python program to fulfil the functionalities of the application choose in the first part of the assignment. The application has an interface designed to enable the user to:

  • Enter at least five records
  • Delete and search records based on keyword/string entry
  • Sort records in a specific order and display them on screen
  • Select an option on the screen to perform a function
  • Get a prompt on screen before an action is performed
